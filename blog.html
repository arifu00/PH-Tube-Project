Discuss the scope of var, let, and const?

=>
Var: var is a function scoped. It's value can be reassigned. Values ​​declared with var can be used anywhere.
Let: Let is a block scoped. It's value can be reassigned. It will give us an error if we try to access it before the declaration.
Const: Const is a block scoped. It's value cannot be reassigned. It will give us an error if we try to access it before the declaration.

Tell us the use cases of null and undefined

=>
Null: It is used to represent the absence of any object or value. It is often set as a value for variables or properties to indicate that they have no value.
Undefined: It indicates a variable that has been declared but has not assigned a value.
